## Per-UI flow (brief)

1. Open a new chat with your chosen aggregator model.
2. Paste `prompt-bundle.txt` (or upload if the UI supports it).
3. Copy the JSON response to a local file, e.g. `~/Downloads/aggregated.json`.
4. Ingest:
   ```
   npm run ingest-aggregated -- \
     --candidate test-omega --version 2027-11-01 \
     --mode manual-webchat \
     --provider <anthropic|openai|google|...> \
     --attested-version "<exact model string from UI>" \
     --attested-by "<your handle>" \
     --file ~/Downloads/aggregated.json
   ```

See `docs/specs/data-pipeline/analysis-modes.md` for the full flow
and review requirements.
