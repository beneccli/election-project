## Step-by-step per chat UI

### ChatGPT (chat.openai.com)

1. Open a new conversation with your chosen model (e.g. GPT-5 thinking).
2. Upload `sources.md` via the attach button (preferred) OR include it
   inline — the bundle already contains it.
3. Paste the contents of `prompt-bundle.txt`.
4. Wait for the JSON response. If the model wraps it in ```json fences,
   the ingest script will strip one layer automatically.
5. Copy the JSON to a local file (e.g. `~/Downloads/chatgpt.json`).
6. Ingest:
   ```
   npm run ingest-raw-output -- \
     --candidate test-omega --version 2027-11-01 \
     --model <model-slug> \
     --mode manual-webchat \
     --provider openai \
     --attested-version "<exact model string from UI>" \
     --attested-by "<your handle>" \
     --file ~/Downloads/chatgpt.json
   ```

## Troubleshooting

- **Validation failure on ingest:** re-run the prompt in the chat,
  ask the model to emit strictly valid JSON without commentary. Do
  not hand-edit the output file.
- **Model returned truncated JSON:** ask the chat UI to continue; copy
  the concatenated full response.
- **You can't remember which model you ran:** rerun. Attestation must
  match reality.
